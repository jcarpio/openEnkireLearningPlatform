# Contribuir a Enkire 🌿

¡Gracias por tu interés en contribuir! Enkire es un proyecto open source para formadores y alumnos.

## Cómo contribuir

### 1. Fork y clone

```bash
git clone https://github.com/TU_USUARIO/enkire.git
cd enkire
npm install
npm run dev
```

### 2. Crea una rama

```bash
git checkout -b feat/nombre-de-tu-feature
# o
git checkout -b fix/nombre-del-bug
```

### 3. Haz tus cambios

- Sigue el estilo de código existente (TypeScript estricto)
- Añade comentarios donde sea necesario
- Actualiza el README si añades features nuevas

### 4. Commit y Push

```bash
git add .
git commit -m "feat: descripción corta del cambio"
git push origin feat/nombre-de-tu-feature
```

### 5. Abre un Pull Request

Describe claramente qué has cambiado y por qué.

---

## Convenciones de commits

Usamos [Conventional Commits](https://conventionalcommits.org/):

| Prefijo | Uso |
|---|---|
| `feat:` | Nueva funcionalidad |
| `fix:` | Corrección de bug |
| `docs:` | Documentación |
| `style:` | Cambios de estilo/CSS |
| `refactor:` | Refactorización sin cambio funcional |
| `chore:` | Mantenimiento, dependencias |

---

## Áreas donde puedes ayudar

- 🌍 **Traducciones** — internacionalización (i18n)
- 🎨 **Diseño** — mejoras de UI/UX
- 📚 **Documentación** — guías para formadores
- 🐛 **Bugs** — reporta y corrige errores
- ✨ **Features** — propón y desarrolla funcionalidades
- 🧪 **Tests** — añade cobertura de tests

---

## Código de conducta

Sé respetuoso, constructivo y generoso. La comunidad Enkire acoge a formadores y alumnos de todos los niveles y orígenes.
